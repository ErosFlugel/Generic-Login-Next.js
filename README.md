## Getting Started

Clona el repositorio:

```bash
git clone https://github.com/ErosFlugel/Generic-Login-Next.js.git
```

Copia el archivo .env:

```bash

```

Instala las dependencias:

```bash
npm i
```

Corre la app:

```bash
npm run dev
```
